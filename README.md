### 開始 ✨

```bash
git clone https://github.com/zhong1016/feastogether.git
cd ./feastogether
```

修改 config.ini 相關資料

Docker :

```bash
docker build -t feastogether .
docker run --rm -it feastogether
```

Local :

下載相關依賴

```bash
go mod download
```

執行程式 :

```bash
go run main.go
```

### 目錄結構 ✨

```text
- client        # booking 程式
- config        # 讀取 ini
- fetch         # 封裝 Http request
- main.go       # 專案入口
- config.ini    # 用戶與餐廳資料
- README.md     # 專案說明文檔
```
